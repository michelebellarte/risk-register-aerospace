## 🎯 What it is

A ready-to-use Excel Risk Register designed for aerospace project management. It automates risk scoring, prioritization, and residual risk tracking — so you spend time managing risks, not formatting spreadsheets.

**4 worksheets:**

| Sheet | Purpose |
|---|---|
| **RISK_REGISTER** | Main register with auto-calculated Risk Score, Priority, and Residual Risk |
| **P×I MATRIX** | Probability × Impact reference matrix with color-coded scoring |
| **RESPONSE_GUIDE** | PMBOK 7 response strategies (Avoid / Mitigate / Transfer / Accept / Escalate) with aerospace examples |
| **HOW_TO_USE** | Step-by-step usage guide |

---

## ✨ Features

- **Automatic Risk Score** — Probability × Impact calculated on entry
- **Color-coded priority** — 🔴 HIGH (≥15) / 🟡 MEDIUM (8–14) / 🟢 LOW (<8), updated automatically
- **Residual risk tracking** — shows remaining exposure after mitigation
- **Live summary dashboard** — counts of high/medium/low and open/closed risks
- **PMBOK 7 aligned** — response strategies follow the official standard
- **Zero setup** — open in Excel and start typing

---

## 📥 How to use

1. Download `RiskRegister_AMX200.xlsx`
2. Open in Microsoft Excel
3. Update the project header (row 2)
4. Add your risks in the blue input cells — scores and priority calculate automatically
5. Use the P×I MATRIX sheet to calibrate scoring in team workshops
6. Choose response strategies from the RESPONSE_GUIDE sheet

Full instructions are in the **HOW_TO_USE** sheet inside the file.
